Quel format pour la landing page ?

La structure globale va définir l'expérience de conversion. Quelle approche te parle le plus ?

A

Page standalone complète

Sans navbar ni footer Projectview. Juste la landing + calendrier. Maximise la conversion, zéro distraction. Le prospect ne peut qu'une seule chose : prendre RDV.

→ Recommandé pour une campagne pub (Google Ads, LinkedIn)
Sections : Hero percutant → Social proof → Bénéfices → Calendrier Google → CTA final
B

Page intégrée au site (avec navbar)

Utilise le layout marketing existant avec Navbar + Footer. Le prospect peut naviguer sur le reste du site. Plus de crédibilité via le contexte de marque.

→ Mieux si le trafic vient du site lui-même
S'intègre dans /la-plateforme ou nouvelle route /demo
C

Mini-header uniquement

Logo Projectview visible en haut pour la crédibilité, mais pas de navigation. Un entre-deux qui garde l'identité de marque sans liens de sortie.

→ Compromis idéal pour les deux types de trafic
Branding visible + focus total sur la conversion
